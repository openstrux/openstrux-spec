# MVP Profile

> Status: draft

Defines the mandatory scope for the Openstrux MVP conformance class.

## Included

- [ ] Core rod types (18 basic rods)
- [ ] Type system v0.4 (union types, type paths)
- [ ] Panel shorthand syntax
- [ ] Config inheritance via strux.context
- [ ] Expression shorthand (filter, projection, aggregation, join, sort, split, guard)

## Excluded from MVP

- Hub packaging
- Audit engine
- Beam Python target
- TypeScript target
- CLI
